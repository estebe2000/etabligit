Aide Générale

L'Établi est une application pédagogique pour la gestion de dépôts GitLab. Elle vous permet de :

  • Créer et gérer des dépôts
  • Éditer des fichiers directement dans l'interface
  • Déployer vos sites web via GitLab Pages
  • Collaborer sur des projets

Pour démarrer l'application

Vous pouvez utiliser le lanceur unifié :

python launcher.py

Version production (ports 8099/5099)

Pour la version production :

python launcher_prod.py

Avec Docker (développement)

docker-compose -f docker-compose.unified.yml up unified

Avec Docker (production)

docker-compose -f docker-compose.prod.yml up etabli
