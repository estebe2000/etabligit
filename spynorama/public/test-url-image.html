Test de la visualisation 360° via URL

Cette page vous permet de tester la nouvelle fonctionnalité de visualisation directe d'images panoramiques 360° via URL.

Comment ça marche

Pour visualiser une image panoramique 360° directement, ajoutez simplement le paramètre image à l'URL de l'application, suivi de l'URL de l'image panoramique:

index.html?image=https://exemple.com/image360.jpg
