Test d'intégration de vidéos Podeduc

Cette page démontre l'intégration de vidéos Podeduc en utilisant directement le code iframe fourni par Podeduc.

Exemple d'intégration

Comment utiliser cette fonctionnalité dans Spynorama

  1. Créez un hotspot vidéo dans votre scène
  2. Sélectionnez "Podeduc" comme type de vidéo
  3. Collez le code iframe fourni par Podeduc dans le champ dédié
  4. Ajoutez un titre et une description si nécessaire
  5. Enregistrez le hotspot

Instructions pour obtenir le code iframe depuis Podeduc

  1. Accédez à la vidéo sur Podeduc
  2. Cliquez sur le bouton "Partager"
  3. Sélectionnez l'option "Intégrer"
  4. Copiez le code iframe complet
  5. Collez ce code dans le champ dédié lors de la création du hotspot
